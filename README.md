# Проект: Путешествие по России

## Описание проекта

Это учебный проект о путешествии по России.
Сайт свёрстан под распространённые разрешения экранов, за счёт чего хорошо выглядит на различных устройствах.

## Используемые технологии

В вёрстке сайта широко используется директива @media, элементы сайта "тянутся" под различную ширину экрана. Файловая структура проекта выполнена по БЭМ.
Разметка страницы сделана с использованием гридов и флексбоксов.
